【Mac 用户运行说明】

由于本程序基于跨平台 Python 架构开发，且您使用的是苹果电脑（macOS），为了让您免去手动配置运行环境的烦恼，我们已经为您准备了自动配置脚本。请按照以下步骤运行：

第一步：双击运行配置脚本
直接双击运行文件夹中的 `run_mac.command`。
- 系统会自动检测您的 Mac 是否有 Python3 环境。
- 接着会自动下载安装所需的全部第三方库依赖（仅首次运行需要联网下载，约需几十秒）。
- 配置完成后会自动为您启动 AI 助手。

※ 常见问题解决：
如果双击运行时，Mac 系统弹出提示“无法打开 run_mac.command，因为您没有适当的访问权限”：
这是 macOS 安全机制导致的，只需进行一次性授权即可，步骤非常简单：
1. 打开 Mac 的“终端” (Terminal) 应用（可在右上角聚焦搜索中搜“终端”）。
2. 输入 `chmod +x `（注意后面有一个空格）。
3. 将 `run_mac.command` 文件直接用鼠标拖入终端窗口中（终端会自动识别并填入该文件的物理路径）。
4. 按下回车键（Enter）。
5. 此时重新双击 `run_mac.command` 即可顺利运行！

第二步：配置您自己的 API 密钥
程序成功启动后为纯净版本（不包含任何开发者的密钥与聊天记录）。您只需点击左下角或右上角的“设置”按钮，在设置面板中填入您自己的 API Key 即可永久免费使用！
